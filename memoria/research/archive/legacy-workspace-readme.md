# Agent Memory Benchmark

Benchmark workspace for evaluating agent memory services such as Memoria, Mem0, and Zep.

## Evaluation plan

- [Memoria and competitor benchmark plan](./BENCHMARK_PLAN.md)

## Datasets

- `datasets/locomo/`: LoCoMo long-term conversational memory benchmark.
- `datasets/longmemeval/`: LongMemEval cleaned benchmark variants and oracle contexts.

See each dataset directory for source and license information.
